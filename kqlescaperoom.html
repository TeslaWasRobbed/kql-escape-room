How to play

  1. Open Sentinel → Logs in your training workspace.
  2. Use these watchlists in your queries:
    • _GetWatchlist('SigninLogs_WL') — Sign-in events
    • _GetWatchlist('SecurityEvent_WL') — Windows logons
    • _GetWatchlist('FileAccess_WL') — File access logs
    • _GetWatchlist('Network_WL') — Network events
  3. Write KQL, run in Sentinel, then paste answers here and click Check.

Tip: Watchlists return strings. Cast with todatetime(), tolong() for time/number comparisons.
